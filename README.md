## SS Boilerplate

A rapid development theme based on Bootstrap 3.0

## Installation ##

Install SS.

Copy across all of the files.

Add
```
SiteConfig:
  extensions:
    - BoilerplateConfig
```

to your config.yml file in the `mysite/_config` directory.

Run a `dev/build` and `?flush=all`
